# Psychological and Somatic Self-Perception Are Weakly Coupled in a Sasang Constitutional Questionnaire

This repository contains the complete, reproducible, and organized R analysis pipeline for the **Sasang-derived survey dataset ($N = 983$ participants)**. 

The core scientific finding is that **psychological and somatic self-perceptions form weakly coupled, partially separable modules, with specific embodied bridge pathways (body-shape and postprandial perception) linking the two domains.**

## 1. Core Re-analysis Findings (Summary Table)

Below is the summary of the core statistical results generated by this pipeline, which represents the primary empirical evidence for the weakly coupled modular self-perception architecture:

| Analysis (분석명) | Empirical Value / Metric (수치 결과) | Academic Interpretation (학술적 해석) |
| :--- | :--- | :--- |
| **Sample Size** | $N = 983$ | Adequate sample size for exploratory categorical analysis |
| **Mean Cramer's V** | Psych–Psych: `0.1757`<br>Soma–Soma: `0.1427`<br>Psych–Soma: `0.1227` | Within-domain associations are systematically stronger than cross-domain associations, consistent with weak cross-domain coupling. |
| **Bootstrap Delta V (1)** | $\Delta V_1 = 0.0529$, 95% CI `[0.0364, 0.0689]` | Within-psychological coupling is reliably stronger than cross-domain coupling. |
| **Bootstrap Delta V (2)** | $\Delta V_2 = 0.0199$, 95% CI `[0.0062, 0.0342]` | Within-somatic coupling is directionally consistent but practically small than cross-domain coupling. |
| **Psych LCA** | $K = 2$ Optimal (BIC: `12299.24`) | Psychological items resolve into a binary structure (Active/outward vs. Passive/inward). |
| **Soma LCA** | $K = 2$ Optimal (BIC: `12418.44`) | Somatic items resolve into a binary structure (Non-sensitive vs. SoEum-like). |
| **LCA Mixed Profiles (Modal)** | Obs: **`55.1%`** (95% CI: `[52.0%, 58.3%]`) vs.<br>Exp: `49.6%` (95% CI: `[49.0%, 50.1%]`) | Observed mixed profiles (e.g. active-somatic + passive-somatic) are significantly higher than expected under complete independence (Excess: `5.5%`, 95% CI: `[2.5%, 8.6%]`). |
| **LCA Mixed Profiles (Weighted)**| Obs: **`55.2%`** (95% CI: `[53.1%, 57.3%]`) | Continuous posterior-probability weighted bootstrap confirms mixed profile predominance, ruling out modal classification uncertainty bias. |
| **Combined 10-item LCA** | $K = 3$ Optimal (BIC: `24669.50`) | Modeling all 10 items jointly requires more classes, indicating that the full response pattern is not reducible to a single binary structure. |
| **MGM Bridge Centrality** | $Q4$ Body Shape: `1.432`<br>$Q2$ Postprandial: `0.830`<br>$Q1$ Energy: `0.786` | Body-shape and postprandial perception emerge as prominent cross-domain bridge nodes. |
| **MGM Isolated Node** | $Q8$ Symptom Response: `0.000` | Somatic illness-symptom response is conditionally isolated under regularized MGM. |
| **RF Classification Accuracy** | Sasang: `84.3%`<br>Soma: `90.4%`<br>E/I: `79.1%` | Baseline prediction accuracies. |

## 2. Repository Layout

This repository utilizes a **zero-clutter architecture** where all intermediate pre-calculated state `.RData` caches and legacy code are isolated within a subfolder, keeping the root directory clean and professional.

```text
.
├── README.md                           # This documentation file
├── analysis_983.csv                    # Clean raw survey dataset (N = 983)
├── analysis_pipeline.R                 # Master script that runs all steps sequentially
│
├── step1_data_cleaning.R               # Cleans and codes data categories into levels
├── step1_schematic.R                   # Generates conceptual domain architecture (Fig 1A)
├── step2_pairwise_association.R        # Computes association matrices (Fig 2A, CSVs)
├── step3_bootstrap_ci.R                # Computes mean Cramer's V & Delta V forest plots (Fig 1B, 1C)
├── step4_domain_lca.R                  # Performs domain-specific poLCA (K = 1 to 6)
├── step5_crosstab_lca.R                # Cross-tabulates LCA modal classes & runs bootstraps (Fig 1D)
├── step6_combined_lca.R                # Performs combined 10-item poLCA (K = 1 to 8)
├── step7_categorical_mgm.R             # Estimates Categorical MGM and bridge strength (Fig 2B, 2C, 2D, S3)
├── step8_supplementary_umap_rf.R       # Run supplementary descriptive UMAP and RF benchmarks (Fig S1, S2)
│
├── cramers_v_groupings.csv             # Pairwise association categories
├── pairwise_cramers_v_10x10.csv        # Symmetric 10x10 Cramer's V association matrix
├── pairwise_p_adj_10x10.csv            # Symmetric 10x10 FDR-adjusted p-value matrix
└── Fig*.png                            # High-DPI publication figures (Fig 1A to Fig S3)
```

## Data

`analysis_983.csv` contains de-identified responses from 983 adults who 
completed an online Sasang-derived self-report questionnaire. The dataset 
includes only categorical variables (age group, gender, MBTI four-letter 
type, item responses Q1–Q10). No direct identifiers (names, emails, IP 
addresses, exact timestamps, or free-text fields) are included. The dataset 
was de-identified prior to release.

## 3. How to Run

### ① Environment & Package Prerequisites
Developed and tested on R 4.6.0. Key package versions used in the analysis:
- poLCA 1.6.0.1
- mgm 1.2-15
- igraph 2.1.1
- umap 0.2.10
- caret 6.0-94
- randomForest 4.7-1.2

Install required packages:
```R
install.packages(c("readr", "dplyr", "tidyr", "ggplot2", "poLCA", "mgm", "igraph", "umap", "caret", "randomForest"))
```

### ② Run the Entire Pipeline
To clean the data, execute all analysis scripts, generate all figures, export supplementary matrix tables, and print the Table 1 summary table to the console, run:

```bash
Rscript analysis_pipeline.R
```

All figures (`Fig*.png`) and matrix CSVs will be output directly to the root directory.

## 4. Key Figures Outline

*   **Figure 2D (Bridge Node Local Profile)**: Focuses on the specific regularized conditional association weights surrounding the primary bridge node ($Q4$ Body Shape).
*   **Figure S1 (UMAP)**: 2D UMAP projection colored by Sasang constitution.
*   **Figure S2 (Random Forest Accuracy)**: Cross-validated Random Forest prediction scores.
*   **Figure S3 (Regularization Sensitivity Analysis)**: Verification of network bridge node ranks across various EBIC $\gamma$ thresholds.
